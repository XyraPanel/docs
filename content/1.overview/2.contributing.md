XyraPanel is community driven. Contributions of all kinds are welcome.

---

## Standards

- Follow existing code style
- Write tests when applicable
- Update documentation as needed
- Use conventional commits
- Create one branch per issue

## How to Contribute

1. Check existing issues first
2. Open an issue to report bugs or suggest features
3. Fork and clone the repo
4. Create a branch for your changes
5. Implement and test thoroughly
6. Push to your fork and open a PR
7. Address feedback during review
